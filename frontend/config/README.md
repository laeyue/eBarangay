# Configuration Files

This folder contains all build and tooling configuration files for the project.

## Files

- **vite.config.ts** - Vite build configuration (dev server, plugins, aliases)
- **tailwind.config.ts** - Tailwind CSS configuration (theme, colors, plugins)
- **postcss.config.js** - PostCSS configuration (autoprefixer, tailwindcss)
- **eslint.config.js** - ESLint linting rules and configuration
- **components.json** - shadcn/ui component configuration

## Usage

These files are automatically loaded by their respective tools. No manual imports needed.
